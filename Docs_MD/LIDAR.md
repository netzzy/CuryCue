# LIDAR

TouchDesigner has LIDAR (Light Detection and Ranging) support directly with specific node types:   
  
  * [Ouster](<./Ouster.md> "Ouster") \- [Ouster](<https://www.ouster.io>) makes LIDAR devices for scanning 3D environments with lasers.
* [Orbbec](<./Orbbec.md> "Orbbec") \- [Orbbec](<https://www.orbbec.com/>) is a developer and manufacturer of 3D cameras and other sensors that can be used within TouchDesigner. The Orbbec Femto Mega and Femto Bolt use the same sensor hardware as the discontinued Kinect Azure and support similar features including 3D body tracking when used in the [Kinect Azure TOP](<./Kinect_Azure_TOP.md> "Kinect Azure TOP").
* [Kinect Azure](<./Kinect_Azure_TOP.md> "Kinect Azure TOP") captures data from a Microsoft Kinect Azure camera or a Kinect-compatible [Orbbec](<./Orbbec.md> "Orbbec") Camera (Femto Mega, Femto Bolt, etc).
* [SICK](<./SICK.md> "SICK") \- makes lidar sensors, primarily for industrial automation applications.
* The [Hokuyo CHOP](<./Hokuyo_CHOP.md> "Hokuyo CHOP") is used for communication with Hokuyo laser scanners (serial or ethernet interface): [Hokuyo Products](<http://www.hokuyo-aut.jp/search/>)
* the [Leuze ROD4 CHOP](<./Leuze_ROD4_CHOP.md> "Leuze ROD4 CHOP")
* the general 2D [Blob Track TOP](<./Blob_Track_TOP.md> "Blob Track TOP")


You can also use other LIDAR devices from other manufacturers with python or ither APIs. 
* [RPLiDAR S2E - ToF LiDAR 360° Laser Range Scanner](<https://www.dfrobot.com/product-2618.html>) from SLAMTEC (30m, Ethernet UDP, IP65)
