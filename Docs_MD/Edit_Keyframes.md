# Edit Keyframes

This will be changing slightly.   
  
You create a [Animation COMP](<./Animation_COMP.md> "Animation COMP"), then right-click on it and select Edit Animation... 

Use the [Animation Editor](<./Animation_Editor.md> "Animation Editor") to add and edit channels and keyframes. 

Then [Export](<./Export.md> "Export") the channels to parameters.
