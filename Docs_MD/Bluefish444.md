# Bluefish444

TouchDesigner supports [Bluefish444](<http://bluefish444.com>) video input and output devices through the [Video Device In TOP](<./Video_Device_In_TOP.md> "Video Device In TOP") and [Video Device Out TOP](<./Video_Device_Out_TOP.md> "Video Device Out TOP").   
  
The KRONOS range is the latest generation of video, audio, and data processing I/O cards from Bluefish444. Built with multi-channel 4K/UHD 60p, High Dynamic Range, and High Frame-Rate SDI to address applications in broadcast, production, post-production, immersive graphics, live events, streaming and more. 

### 

Bluefish Support in TouchDesigner

TouchDesigner has built-in native driver support for Bluefish devices. 

**Ways to interface with Bluefish444 cards in TouchDesigner**
* [Video Device In TOP](<./Video_Device_In_TOP.md> "Video Device In TOP") \- Captures video streams from capture cards and input devices.
* [Video Device Out TOP](<./Video_Device_Out_TOP.md> "Video Device Out TOP") \- Outputs video.
