# AV1

[AV1](<https://en.wikipedia.org/wiki/AV1>) ([AOMedia Video 1](<https://en.wikipedia.org/wiki/AV1>) codec) is an open, royalty-free video coding format initially designed for video transmissions over the Internet. It was developed by the [Alliance for Open Media](<https://en.wikipedia.org/wiki/Alliance_for_Open_Media>) (AOMedia). 

AV1 is read by the [Movie File In TOP](<./Movie_File_In_TOP.md> "Movie File In TOP") and [Video Stream In TOP](<./Video_Stream_In_TOP.md> "Video Stream In TOP"). It is output by the [Movie File Out TOP](<./Movie_File_Out_TOP.md> "Movie File Out TOP") and [Video Stream Out TOP](<./Video_Stream_Out_TOP.md> "Video Stream Out TOP").
