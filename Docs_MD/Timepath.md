# Timepath

The Timepath of a node or network is the path to the hierarchical [Component Time](<./Component_Time.md> "Component Time") that governs that node or network's time-based attributes.   
  
The base time at the root of the TouchDesigner network hierarchy is called [Root Time](<./Root_Time.md> "Root Time"), the Timepath for root time = / 

The Timepath for any node or network in TouchDesigner can be determined using the Tscript`Timepath()`expression.
