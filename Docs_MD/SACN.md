# SACN

sACN (Streaming Architecture for Control Networks) is a [DMX](<./DMX.md> "DMX") protocol over IP networks using UDP. Unlike DMX over XLR cable with up to 256 universes, sACN allows for up to 64,00 universes. 

sACN reduces bottlenecks as it can operate in MultiCast mode. Furthermore, the IP addresses can be all different, as long as they subscribe to an accessible host IP and port. 

sACN is described more completely on the [Artistic License Integration sACN](<http://artisticlicenceintegration.com/?page_id=2600>) page. 

sACN is supported in the [DMX Out CHOP](<./DMX_Out_CHOP.md> "DMX Out CHOP") and [DMX In CHOP](<./DMX_In_CHOP.md> "DMX In CHOP").
