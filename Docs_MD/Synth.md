# Synth

**Synths** is a legacy term for the artworks created by TouchDesigner. A Synth consists of the [.toe](<./.md> ".toe") file created by TouchDesigner and all the associates media files that are needed to run an artwork in [TouchPlayer](<./TouchPlayer.md> "TouchPlayer") or, in [Perform Mode](<./Perform_Mode.md> "Perform Mode"), [TouchDesigner](<./TouchDesigner.md> "TouchDesigner"). 

An archive of TouchDesigner 017 synths (circa 2007) is available [here](<http://www.derivativeinc.com/Artworks/Catalog.asp?page=NewReleasesRetrieveRandomOther.asp>). (link broken, contact Derivative)
