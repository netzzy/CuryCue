# Generative Design

## 

Summary

This is a collection of examples from the book [Generative Design](<http://www.generative-gestaltung.de/1/>) converted to TouchDesigner. The examples highlight techniques that cover almost all aspects of TouchDesigner and should be a great resource for learning and further understanding the software. As of publishing this collection holds about 088 examples. The website accompanying the book has all the original code available [here](<http://www.generative-gestaltung.de/1/code>). 

Many thanks to the creators of the original [Processing](<http://processing.org>) based sketches.
