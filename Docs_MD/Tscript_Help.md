# Tscript Help

There are two scripting languages in TouchDesigner: [Python](<./Python.md> "Python") and [Tscript](<./Tscript.md> "Tscript"). Tscript Commands and Expression help can be found in the [Commands](<./Tscript_Commands.md> "Tscript Commands") and [ Expressions](<./Tscript_Expressions.md> "Tscript Expressions") categories. 

## Via the Textport

You can also browse commands and expressions via the textport: 

**help** gives you a list of all Tscript commands and 

**exhelp** gives you a list of all Expressions 

Append the **-k** flag to the above commands to search for commands containing the word you are looking for. For example, to find all the expressions whose help contains the word "chop" type: 

**exhelp -k chop**

See Also: [Commands](<./Tscript_Commands.md> "Tscript Commands"), [ Expressions](<./Tscript_Expressions.md> "Tscript Expressions")
