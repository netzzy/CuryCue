# Units

The units menu on the right of many CHOP parameters lets you specify the parameter in the following ways: 
* Index (the sample number in the CHOP, starting at 0, even for time slices)
  * Frames (index = global frames per second * parameter value). global fps is default 60.
  * Seconds (index = CHOP sample rate * parameter value)


Units are treated more extensively in the [Animation component](<./Animation_COMP.md> "Animation COMP") to include user-defined units.
