# Palette:autoMediaPlayer

##   
  
Summary

The autoMediaPlayer allows for looping over movies and still images in a specified folder on disc in linear or randomized order. There are options to adjust the output resolution, change the length of still images, as well as set the transition length. 

[![PythonIcon.png](./images/c/c2/PythonIcon.png)](</File:PythonIcon.png>)[Palette:autoMediaPlayer Ext](</index.php?title=Palette:autoMediaPlayer_Ext&action=edit&redlink=1> "Palette:autoMediaPlayer Ext \(page does not exist\)")

## 

Parameters - Auto Media Player Page

Media Folder`Mediafolder`\- Select the folder that holds all the movies and still images to display. 

Current File`Currentfile`\- Displays the currently playing file. 

Randomize Playback`Togglerandom`\- Turn on to randomize playback order. 

Transition Length (sec)`Crosslength`\- Set the time in seconds for the transition between mediafiles. 

Filter Curve`Filtertype`\- ⊞ \- Select between a linear or gaussian crossfade curve. 
* Gaussian`gauss`-
* Linear`box`-


Hold Duration of Images (sec)`Imagelength`\- Specify the number of seconds a still image should be displayed. 

Output Resolution`Outputres`\- ⊞ \- Specify the output resolution. 
* Output Resolution`Outputresx`-
* Output Resolution`Outputresy`-


Fit Method`Fitmethod`\- ⊞ \- Specify how playing mediafiles are fit to the output resolution. 
* Fill`fill`-
* Fit Horizontal`fithorz`-
* Fit Vertical`fitvert`-
* Fit Best`fitbest`-
* Fit Outside`fitoutside`-
* Native Resolution`nativeres`-


Reset`Reset`\- Restart the playback.
