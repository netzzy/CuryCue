# TouchDesigner Pro

## **TouchDesigner Pro**

TouchDesigner Pro includes everything TouchDesigner has to offer. 

### Pro Support

Purchasing a Pro license gets you into our [Pro Support](<./Pro_Support.md> "Pro Support") program. Our team is available to help you with any programming, production, or consulting needs you have. Use our years of experience to make your life easier. 

Each Pro license includes 6 hours of Pro Support. Additional hours can be purchased for larger tasks and additional support. Please read [Pro Support](<./Pro_Support.md> "Pro Support") for more details on what the program includes. 

### Features Unlocked
* [Notch](<./Notch.md> "Notch") operators without any resolution limits (requires Notch license)
  * 3rd party Projection calibration via [Vioso TOP](<./Vioso_TOP.md> "Vioso TOP") or [Scalable Display TOP](<./Scalable_Display_TOP.md> "Scalable Display TOP")
  * [Stype](<./Stype.md> "Stype") optical tracking for cameras
  * for tracking data from [BlackTrax Motion Tracking Systems](<http://blacktrax.cast-soft.com/>): [BlackTrax CHOP](<./BlackTrax_CHOP.md> "BlackTrax CHOP")
  * sync multiple processes or systems with [Sync In CHOP](<./Sync_In_CHOP.md> "Sync In CHOP") and [Sync Out CHOP](<./Sync_Out_CHOP.md> "Sync Out CHOP")
  * hardware frame-lock ([NVIDIA Gsync](<http://www.nvidia.com/object/nvidia-sync-quadro-gsync.html>) / [AMD S400](<http://www.amd.com/us/products/workstation/graphics/s400/Pages/s400.aspx>) sync cards)
  * [Privacy Options](<./Privacy.md> "Privacy") for private .toe files and private components. Create a`[.toe](<./.md> ".toe")`project file or`[.tox](<./.md> ".tox")`component file that is inaccessible without the password. Additionally you can create components that only run with USB License Dongle. See [Privacy](<./Privacy.md> "Privacy") for more details.
  * for blending motion channels: [Clip Blender CHOP](<./Clip_Blender_CHOP.md> "Clip Blender CHOP"), [Clip CHOP](<./Clip_CHOP.md> "Clip CHOP"), [Clip DAT](<./Clip_DAT.md> "Clip DAT")
  * bug fixes to the specific build you are using
  * software fixes/RFEs are not guaranteed, but Pro users will receive front-of-the-line service.
  * [Pro Support](<./Pro_Support.md> "Pro Support") program offering direct access to Derivative's programmers and product specialists.


For a comparison of features, see [Licensing](<./Licensing.md> "Licensing").
