# Parent

In the network path hierarchy, the **parent** of a node or network is said to be the node or network in which the node is located. So in the network`/project1/test/notes`, the`test`node/network is said to be the parent of`notes`. 

In [3D Parenting](<./3D_Parenting.md> "3D Parenting"), a node is attached to another node in the same network by their top/bottim connectors, but also can be oarented by being inside another node. 

In 2D control panels, a [Panel Component](<./Panel_Component.md> "Panel Component") is displayed as within another panel by (again), a node is attached to another node in the same network like in [3D Parenting](<./3D_Parenting.md> "3D Parenting"), but also can be parented by being inside another Panel component.
