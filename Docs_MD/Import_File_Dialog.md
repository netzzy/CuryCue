# Import File Dialog

## Description

Import any asset readable by TouchDesigner. See [File Types](<./File_Types.md> "File Types") for a summary all readable file formats. 

## Importing a File

In the **File** menu, select **Import File...**. Alternatively use the keyboard shortcut ctrl+i ([macOS](<./MacOS.md> "MacOS") cmd+i). 

A file browser will open, select the file to import and click **Open**. 

The file will be loaded into the network pane last used, ie. whichever network pane currently has focus. The [Status Bar](<./Status_Bar.md> "Status Bar") will confirm the operation was successful and the network path the asset was imported to.
