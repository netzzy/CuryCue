# Parameter Expression

A parameter expression is a Python expression that is in the expression field of a parameter. This can be a simple math equation or a complex expression using [Python](<./Python.md> "Python"). Some common python expressions in parameters are in [Python Tips](<./Python_Tips.md> "Python Tips").   
  
For help with Python, refer to [Python](<./Category-Python.md> "Category:Python"). 

**Tip** : Long expressions in parameters can be edited in your text editor by clicking in the expression and pressing Ctrl-e. You can also select Edit Expression in the parameter's right-click menu. 

( Nodes can be put into the legacy expression format, [Tscript](<./Tscript.md> "Tscript"). For a list of all Tscript expressions, refer to [Tscript Expressions](<./Tscript_Expressions.md> "Tscript Expressions") or open the [textport](<./Textport.md> "Textport"), put it in tscript mode by clicking the language option in the upper left, then type`exhelp`. )
