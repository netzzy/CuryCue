# Learn TouchDesigner

## Getting Started
* The **[Getting Started Tips](<./Getting_Started_With_TouchDesigner.md> "Getting Started With TouchDesigner")** will help you [install TouchDesigner](<./Install_TouchDesigner.md> "Install TouchDesigner") and get it keyed and running.
  * The **[TouchDesigner Curriculum](<http://learn.derivative.ca>)** course is where you should start to learn the fundamentals of TouchDesigner.
  * Check out both the **[Beginner - Interactive and Immersive HQ playlist](<https://youtube.com/playlist?list=PLSqkC3f_BStyl5j6NqYwLOErW0ych_4k1&si=auI8NiNZbfnqvirH>)** and **[Beginner - Bileam Tschepe playlist](<https://youtube.com/playlist?list=PLSqkC3f_BStyHGlnu0OnnXAWU9PWDokOV&si=TgcDBM5UaqEeihN5>)** in the community playlists section.
  * **[Introduction to TouchDesigner](<https://www.youtube.com/watch?v=wmM1lCWtn6o&t=150s>)** workshop highlights many concepts and specific operators through introductory examples in a single 4 hour workshop.
  * Go through **[First Things to Know about TouchDesigner](<./First_Things_to_Know_about_TouchDesigner.md> "First Things to Know about TouchDesigner")** to get an introductory overview with pro tips and tricks throughout.

## Documentation
* **[Search](<https://docs.derivative.ca/Special:Search>)** documentation using the wiki advanced search page.
  * The **[TouchDesigner Glossary](<./TouchDesigner_Glossary.md> "TouchDesigner Glossary")** is the gateway to the concepts and architecture of TouchDesigner.
  * Browse the **[Categories](<https://docs.derivative.ca/Special:Categories>)** found in the navigation bar on the left.
  * On the splash page when starting TouchDesigner, press **TOUCHTIPS** to get various usage tips while-u-wait.
  * **[Operator Help pages](<./Operator.md> "Operator")** \- All the node types of TouchDesigner. And check out OP Snippets below.
  * **[Python](<./Category-Python.md> "Category:Python")** page is the main jumping off point for all things Python in TouchDesigner. **[Python Reference](<./Category-Python_Reference.md> "Category:Python Reference")** is an alphabetical index of all Python pages in the wiki. **[Python Classes and Modules](<./Python_Classes_and_Modules.md> "Python Classes and Modules")** is grouped by subject. Also see the **[Introduction to Python Tutorial](<./Introduction_to_Python_Tutorial.md> "Introduction to Python Tutorial")** to learn the basics of Python scripting in TouchDesigner.
  * **Help - > Python Examples** in the TouchDesigner UI takes you to a`PythonExamples.toe`file with 100+ examples.
  * **[Offline Help](<./Offline_Help.md> "Offline Help")** \- Access the wiki when you are not connected to the internet by pre-downloading the offline documentation package.

## Sample Components, Tools and Examples
* **[OP Snippets](<./OP_Snippets.md> "OP Snippets")** \- Over 1000 functioning examples found via (1) the Help -> [Operator Snippets](<./OP_Snippets.md> "OP Snippets") menu, or (2) right-click on any operator in any network, or (3) right-click on any operator in the OP Create menu. When you get there, browse the various OP types, or try pressing Rand or Newest.
  * **[Palette Browser](<./Palette.md> "Palette")** \- In the menu Dialogs -> Palette Library, or the left-pane of the default TouchDesigner. For a list of all Palette component help pages, see [Category:Palette](<./Category-Palette.md> "Category:Palette").
  * **[Community Assets](<https://tinyurl.com/bde5arht>)** \- User-contributed components and assets for TouchDesigner in our website's community feed. If you'd like to share an asset, please upload it here by logging into [Derivative](<https://derivative.ca/>) and creating an ASSET under **My Account**..
  * **[Olib from alphamoonbase](<https://olib.amb-service.net/>)** \- A highly-recommended growing and active resource of free TouchDesigner Components.
  * **ask google** \- just type “touchdesigner how …” and see what you get - example - [touchdesigner how hand track to sound](<https://www.google.com/search?q=touchdesigner+how+hand+track+to+sound&ie=UTF-8&oe=UTF-8&hl=en-ca>)
