# Getting Started With TouchDesigner

###   
  
Getting Started with TouchDesigner

Make sure your computer conforms with the minimal [System_Requirements](<./System_Requirements.md> "System Requirements") and get a 3-button mouse. 

On Windows, update your computer's graphic drivers and apply any Windows Updates available. On macOS, update to the most recent update of macOS available in your macOS version. ie if you are on macOS 13, update to the most recent 13.x. 

To download TouchDesigner, go to **[Downloads](<https://derivative.ca/download>)**

While downloading, [create an account](<https://www.derivative.ca/Login/RegisterForm.asp>) so you can get a key after installation. 

[Install](<./Install_TouchDesigner.md> "Install TouchDesigner") and start TouchDesigner. 

### 

Licensing

Start TouchDesigner, the Key Manager Dialog will open if no key is present. [Create a key](<./Key_Manager_Dialog.md> "Key Manager Dialog"). 

[TouchDesigner Non-Commercial](<./TouchDesigner_Non-Commercial.md> "TouchDesigner Non-Commercial") can be used (free) for personal and educational use, including demoing and learning how to use TouchDesigner. 

[Educational](<./TouchDesigner_Educational.md> "TouchDesigner Educational"), [Commercial](<./TouchDesigner_Commercial.md> "TouchDesigner Commercial") and [Pro](<./TouchDesigner_Pro.md> "TouchDesigner Pro") licenses are available from the [store](<https://www.derivative.ca/shop>), the [Licensing](<./Licensing.md> "Licensing") page explains the differences in features. 

### 

Where to Start Learning TouchDesigner

Check the official [TouchDesigner Curriculum](<https://learn.derivative.ca>) for an introduction to the fundamentals. 

The [Learn TouchDesigner](<./Learn_TouchDesigner.md> "Learn TouchDesigner") page will introduce you to all the learning materials available for TouchDesigner, including tutorials, examples, and documentation. 

If you are interested in attending workshops to learn TouchDesigner, check out [Workshops & Events](<https://derivative.ca/workshops-events>). 

For those eager to write some code, Python is the default scripting language of TouchDesigner. You can find all the Python documentation and reference material starting here: [Python Reference](<./Category-Python.md>).
