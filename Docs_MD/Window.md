# Window

A Window in TouchDesigner is a window in Microsoft Windows or [macOS](<https://developer.apple.com/design/human-interface-guidelines/macos/windows-and-views/window-anatomy/>) that contains either (1) the TouchDesigner editing interface that exists in [Designer Mode](<./Designer_Mode.md> "Designer Mode"), or (2) a user-created [Panel](<./Panel.md> "Panel") inside a [Window Component](<./Window_COMP.md> "Window COMP"). The user-created windows can span [Multiple Monitors](<./Multiple_Monitors.md> "Multiple Monitors") borderless, or be floating windows with borders, or popups. 

See also [Window Placement Dialog](<./Window_Placement_Dialog.md> "Window Placement Dialog"), [Window COMP](<./Window_COMP.md> "Window COMP"), [Multiple Monitors](<./Multiple_Monitors.md> "Multiple Monitors").
