# FFmpeg

FFmpeg is a software library for reading and writing a wide range of image, movie and audio files. It is used by the [Movie File In TOP](<./Movie_File_In_TOP.md> "Movie File In TOP") and the [Movie File Out TOP](<./Movie_File_Out_TOP.md> "Movie File Out TOP") as well as other audio and video reading/writing nodes.`ffmpeg.exe`is included in TouchDesigner's`bin`folder. TouchDesigner [TOPs](<./TOP.md> "TOP") do most of what`ffmpeg.exe`does. Where they do not, one of many sites with`ffmpeg.exe`examples is [labnol.org](<https://www.labnol.org/internet/useful-ffmpeg-commands/28490/>)

See also [FFmpeg.org](<https://www.ffmpeg.org/>), [FFmpeg wikipedia](<https://en.wikipedia.org/wiki/FFmpeg>), [File Types](<./File_Types.htm#Files_Imported> "File Types")
