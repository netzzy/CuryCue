# Slug Library

[![Slug.png](./images/thumb/b/bd/Slug.png/300px-Slug.png)](</File:Slug.png>)  
  
For typography and font rendering in TouchDesigner, the [Slug Font Rendering Library](<https://sluglibrary.com/>) is a dynamic GPU font rendering and layout engine for professional high-quality, resolution-independent text. You can render fonts with the precision required to match industry-standard typography tools and ensure your font and text designs can be faithfully recreated in TouchDesigner. Supporting all [Unicode](<./Unicode.md> "Unicode") characters, it is fully scalable from small to huge sizes with sharp outlines and no artifacts at any resolutions. 

## Text-Generating Operators

The [Text COMP](<./Text_COMP.md> "Text COMP") uses Slug to render resolution-independent 2D text and icons. 

The [Geo Text COMP](<./Geo_Text_COMP.md> "Geo Text COMP") uses Slug to render text and icons in 3D renders. 

The [Text TOP](<./Text_TOP.md> "Text TOP") renders rasters at specific resolutions, using the Display Method called 'Scalable' and is the default rendering method used when the Text TOP is set to 'Automatic' and the font size is 16 or larger.. 

See also [Text COMP](<./Text_COMP.md> "Text COMP"), [Geo Text COMP](<./Geo_Text_COMP.md> "Geo Text COMP"), [Text TOP](<./Text_TOP.md> "Text TOP")
