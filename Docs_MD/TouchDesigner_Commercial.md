# TouchDesigner Commercial

## **TouchDesigner Educational and Commercial**

TouchDesigner Educational and Commercial have identical feature sets, they differ only in their intended use-case. 

### Educational

Available to students and schools for **non-commercial and educational uses only**. (Precisely, "You are not receiving money or compensation for your work with TouchDesigner") 

Students and school administrators must register for an Educational license and be approved by Derivative before being able to purchase the license. [Register Here](<https://www.derivative.ca/product/touchdesigner-educational>)
* Regular Educational license $300 USD per license. Can be used with [Key Manager Dialog](<./Key_Manager_Dialog.md> "Key Manager Dialog") or [License Dongle](<./License_Dongle.md> "License Dongle") (not included)
  * [Floating Cloud](<./Floating_Cloud_Licenses.md> "Floating Cloud Licenses") Commercial license $450 USD per license
  * Purchase TouchDesigner Educational in the [STORE](<https://www.derivative.ca/product/touchdesigner-educational>).
  * [Volume Discounts](<./Volume_Discount.md> "Volume Discount") are available for TouchDesigner Educational.


See also [Educational Setup](<./Educational_Setup.md> "Educational Setup") for more information about setting up TouchDesigner at educational institutions. 

### Commercial

A Commercial (or [Pro](<./TouchDesigner_Pro.md> "TouchDesigner Pro")) license is **required for any project or work done with TouchDesigner that you receive money or compensation for**. 

Some examples include a) you are working on a paying project, b) you use TouchDesigner to promote your work to create income, c) you or a team use multiple computers to create parts of a larger commercial project, in all these cases a Commercial license is required for all computers that are used on these projects. 
* Regular Commercial license $600 USD per license. Can be used with [Key Manager Dialog](<./Key_Manager_Dialog.md> "Key Manager Dialog") or [License Dongle](<./License_Dongle.md> "License Dongle") (not included)
  * [Floating Cloud](<./Floating_Cloud_Licenses.md> "Floating Cloud Licenses") Commercial license $900 USD per license
  * Purchase TouchDesigner Commercial in the [STORE](<https://www.derivative.ca/product/touchdesigner-commercial>).

### Features Unlocked
* image resolution and compositing at any resolution the hardware supports
  * create H.264/H.265 movies in realtime (requires Nvidia GPU)
  * stream textures via RTSP, RTMP, or SRT using the [Video Stream Out TOP](<./Video_Stream_Out_TOP.md> "Video Stream Out TOP") (requires Nvidia GPU)
  * Up to 2 [Notch](<./Notch.md> "Notch") operators running at 1920x1080 (requires Notch license)
  * LIDAR scanner [Ouster TOP](<./Ouster_TOP.md> "Ouster TOP")
  * laser scanner [Leuze ROD4 CHOP](<./Leuze_ROD4_CHOP.md> "Leuze ROD4 CHOP")
  * all [Shared Memory](<./Shared_Memory.md> "Shared Memory") operators
  * [send](<./DirectX_Out_TOP.md> "DirectX Out TOP") and [receive](<./DirectX_In_TOP.md> "DirectX In TOP") texture to DirectX applications
  * use [TouchEngine](<./TouchEngine.md> "TouchEngine") in other applications
  * option to hide the TouchDesigner splash screen on startup
  * the Commercial license is good for one commercial key for one computer
  * keys can be moved from computer to computer
  * does not include any direct support, [forum](<https://forum.derivative.ca>) support only


See also [TouchDesigner Pro](<./TouchDesigner_Pro.md> "TouchDesigner Pro")
