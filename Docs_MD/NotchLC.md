# NotchLC

[![NotchLC white+orange.png](./images/thumb/7/7b/NotchLC_white+orange.png/300px-NotchLC_white+orange.png)](</File:NotchLC_white%2Borange.png>)

[](</File:NotchLC_white%2Borange.png> "Enlarge")

NotchLC (more information [here](<https://notchlc.notch.one/>)) is a high quality, GPU accelerated video format. It offers higher quality results than HapQ, at the cost of higher GPU usage as well as larger file sizes. It also provides 10-bit color output. 

The [Movie File In TOP](<./Movie_File_In_TOP.md> "Movie File In TOP") reads and decodes NotchLC movie files while the [Movie File Out TOP](<./Movie_File_Out_TOP.md> "Movie File Out TOP") encodes NotchLC codec into movie files. 

The 'Quality' slider in the [Movie File Out TOP](<./Movie_File_Out_TOP.md> "Movie File Out TOP") linearly blends between the various NotchLC quality settings. A quality of 1.0 is the same as NotchLC's 'Best' quality, and 0.0 is the same as NotchLC's 'Good' quality. Matching up with the other quality levels can be achieved by using the following values: 

Quality Settings  NotchLC Quality Name | Quality Parameter Value  
---|---  
Good | 0.0   
Good Very | 0.285   
Excellent | 0.575   
Optimal | 0.862   
Best | 1.0
