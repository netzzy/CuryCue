# RMB Menu

The **RMB Menu** or **Context Menu** is the menu that appears when the right mouse button is clicked on an [Operator](<./Operator.md> "Operator").   
  
See [Mouse Click](<./Mouse_Click.md> "Mouse Click"). 

## Common Options
* Parameters 
    * Reset to Defaults
  * Viewer
  * Edit Comment
  * Create Movie
  * Explore Media Folder
  * Change OP Type
  * Select Inputs 
    * Outputs
  * Force Cook
  * Delete
