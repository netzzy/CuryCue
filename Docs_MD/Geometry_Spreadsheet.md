# Geometry Spreadsheet

The **Geometry Spreadsheet** was one of TouchDesigner's [Pane](<./Pane.md> "Pane") Types. Change to Geometry Spreadsheet by using the Pane Type Menu or using the **Alt+7** shortcut. 

Currently, use a [SOP to DAT](<./SOP_to_DAT.md> "SOP to DAT") to look at SOP data values. It shows detailed information about Points or [Primitives](<./Primitive.md> "Primitive") in a SOP.
