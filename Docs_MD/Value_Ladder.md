# Value Ladder

Value Ladders allow you to change the value of numerical parameters with a mouse. Hold [MMB](<./Mouse_Click.md> "Mouse Click") (alternatively, hold [LMB](<./Mouse_Click.md> "Mouse Click")) on the parameter's value field or label to open the value ladder. The value ladder will pop up with .01 .1 1 10 100 options. With middle-mouse still down, move to one of the powers of 10, then drag to the right/left to increase/decrease the value by that increment.
