# GLSL

GLSL is the OpenGL Shading Language, used in TouchDesigner to define 3D shaders in [Materials](<./MAT.md> "MAT"), and 2D shaders in [GLSL TOPs](<./GLSL_TOP.md> "GLSL TOP").   
  
See also [Shader](<./Shader.md> "Shader"), [Write a GLSL Material](<./Write_a_GLSL_Material.md> "Write a GLSL Material"), [Write a GLSL TOP](<./Write_a_GLSL_TOP.md> "Write a GLSL TOP"), [GLSL Multi TOP](<./GLSL_Multi_TOP.md> "GLSL Multi TOP"), plus the [GLSL Category](<./Category-GLSL.md> "Category:GLSL") and [Rendering Category](<./Category-Rendering.md> "Category:Rendering").
