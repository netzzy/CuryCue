# Keyboard Shortcuts

There are two types of keyboard shortcuts:   
  
[Application Shortcuts](<./Application_Shortcuts.md> "Application Shortcuts") are built-in to TouchDesigner's authoring interface, which can also be edited and customized. Shortcut is TouchDesigner's term for hot key. 

You can also create [Panel Shortcuts](<./Panel_Shortcuts.md> "Panel Shortcuts") for any custom-built panel that you build with [Panel Components](<./Panel_Component.md> "Panel Component"). 

These should not be confused with [Operator Shortcuts](<./Operator_Shortcuts.md> "Operator Shortcuts").
