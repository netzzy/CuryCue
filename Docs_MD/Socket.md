# Socket.IO

Socket.IO enables real-time, bidirectional and event-based communication. It works on every platform, browser or device, focusing equally on reliability and speed.[[1]](<#cite_note-1>)

A common framework for creating a Socket.IO server app is [Node.js](<https://nodejs.org/>) \- a JavaScript runtime built on Chrome's V8 JavaScript engine. 

See the [SocketIO DAT](<./SocketIO_DAT.md> "SocketIO DAT").
