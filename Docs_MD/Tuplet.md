# Tuplet

A tuplet is the set of [parameters](<./Parameter.md> "Parameter") on one line of the parameter dialog.   
  
For example, on a Transform SOP, a tuplet`t`contains the parameter`tx`,`ty`and`tz`. 

A tuplet can contain one parameter. In this case, the tuplet name is the parameter name. 

Tuplets occupy a [page](<./Page_Class.md> "Page Class") of parameters. 

See also: [Page_Class](<./Page_Class.md> "Page Class")
