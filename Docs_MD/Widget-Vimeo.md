# Widget:Vimeo

This widget allows you to embed **[Vimeo](<http://www.vimeo.com/>)** on your wiki page. 

Created by [Sergey Chernyshev](<http://www.mediawikiwidgets.org/User:Sergey_Chernyshev>)

## Using this widget

For information on how to use this widget, see [widget description page on MediaWikiWidgets.org](<http://www.mediawikiwidgets.org/Vimeo>). 

## Copy to your site

To use this widget on your site, just install [MediaWiki Widgets extension](<http://www.mediawiki.org/wiki/Extension:Widgets>) and copy [full source code](<https://docs.derivative.ca/index.php?title=Widget:Vimeo&action=edit>) of this page to your wiki as **Widget:Vimeo** article.
