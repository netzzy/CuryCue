# Notch

[Notch](<https://www.notch.one/>) is a realtime 3D authoring and rendering package. 

### Notch Block Playback

Notch Block playback is integrated in TouchDesigner via the [Notch TOP](<./Notch_TOP.md> "Notch TOP"), allowing for direct control of Notch blocks inside a TouchDesigner project. This feature is available when using any paid licenses, but Commercial and Educational licenses have limits on their capabilities, as listed [here](<./Licensing.md> "Licensing"). 

Loading a Notch block automatically generates any exposed properties as custom parameters on the Notch TOP. 

A Notch Builder Pro license is required to load the blocks into TouchDesigner without a watermark (see [Notch Builder licensing](<https://www.notch.one/products/notch-builder/>)). With the Notch Builder Pro license, the resolution of the Notch block is limited to 4096x2160, any resolution over that amount will have a watermark. For higher resolutions or multiple HD outputs, a Notch Playback license is required. For Notch Playback licensing and resolution support, please refer to [Notch Playback Licensing](<https://www.notch.one/products/notch-playback/>). 

CodeMeter User Runtime v7.60+ must also be installed: <https://www.wibu.com/support/user/downloads-user-software.html>

### NotchLC Codec

The [NotchLC](<./NotchLC.md> "NotchLC") codec is also supported for both decode and encode. More information available in it's article.
