# Change OP Type Dialog

The Change OP Type dialog lets you change an [operator](<./Operator.md> "Operator") to another type in the same family. [RMB](<./Mouse_Click.md> "Mouse Click") click on any node and select **Change OP Type...** from the pop-up menu, this will open the dialog. Select and click on the new OP type you want to replace the old one with. 

**TIP:** If you select the same OP type, the operator will be deleted and re-created. This is a handy way to recreate an operator if it is in error or you wish to reset all parameters to their default. 

See Also [OP Create Dialog](<./OP_Create_Dialog.md> "OP Create Dialog")
