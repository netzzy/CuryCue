# Category:Geometry

3D Geometry in TouchDesigner is held in [SOPs](<./SOP.md> "SOP"), and is stored in the data structure called a [Geometry Detail](<./Geometry_Detail.md> "Geometry Detail"). The Geometry Detail holds a a [Point List](<./Point_List.md> "Point List"), a [Primitive](<./Primitive.md> "Primitive"), and for each Polygon/Spline-type primitive, it holds a Vertex List. The types of primitives are polygon, mesh, spline, metaballs, spheres, cylinders and particles. 

Geometry is rendered to an image by the [Render TOP](<./Render_TOP.md> "Render TOP"), which takes a set of [Geometry Component](<./Geometry_COMP.md> "Geometry COMP") that contain SOPs whose Render flags are on. 

Here are the details on the types of geometry that TouchDesigner is capable of rendering.
