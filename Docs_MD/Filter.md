# Filter

Operators that need 1 or more inputs (to create a meaningful output) are called Filters in TouchDesigner, like a [Math CHOP](<./Math_CHOP.md> "Math CHOP"). "Inputs" are both the wires from other operators, and references to other operators in OP parameters.   
  
Filters are the lighter shaded operators in the [OP Create Dialog](<./OP_Create_Dialog.md> "OP Create Dialog"). 

See their counterparts [Generators](<./Generator.md> "Generator"). 

Operators may have inputs defined by [Parameters](<./Parameter.md> "Parameter") that are [Path](<./Network_Path.md> "Network Path") references to other operators, that are from any [Operator Family](<./Operator_Family.md> "Operator Family").
