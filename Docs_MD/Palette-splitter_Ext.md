# Palette:splitter Ext

These Extensions reference a specific [Palette:splitter](<./Palette-splitter.md> "Palette:splitter"). The Extension class of a splitter COMP.

## Methods`SplitterExt.Assign(splitId, screen)`: 

> Pass a splitId and a screen (monitor) number to generate an assignment. The UI will update if both the specified splitId and screen exists. The Splitter UV Map will update as well.`SplitterExt.Reset(self)`: 

> When called, Reset will empty the Top parameter if any TOP was specified as well as unassign and delete splits.
