# Designer Mode

In Designer Mode, you edit your [Network](<./Network.md> "Network"), [Nodes](<./Node.md> "Node") and [Parameters](<./Parameter.md> "Parameter") in the [Network Editor](<./Network_Editor.md> "Network Editor"), [Animation Editor](<./Animation_Editor.md> "Animation Editor") and other [Panes](<./Pane.md> "Pane").   
  
In contrast, [Perform Mode](<./Perform_Mode.md> "Perform Mode") is the user interface you create in Designer Mode. 

Designer Mode has its [Application Shortcuts](<./Application_Shortcuts.md> "Application Shortcuts"), distinct from those of Perform Mode. 

See also [Perform Mode](<./Perform_Mode.md> "Perform Mode"), [Window Placement](<./Window_Placement_Dialog.md> "Window Placement Dialog"), [Privacy](<./Privacy.md> "Privacy").
