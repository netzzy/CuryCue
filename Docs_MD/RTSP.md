# RTSP

The [Video Stream Out TOP](<./Video_Stream_Out_TOP.md> "Video Stream Out TOP") can create an RTSP server to send H.264 video and MP3 audio across the network. It uses Nvidia's hardware H264 encoder. For RTSP, it can handle multiple clients connecting to it at the same time. 

The [Video Stream In TOP](<./Video_Stream_In_TOP.md> "Video Stream In TOP") can create an RTSP client to receive video and audio across the network. 

See also [Video Streaming User Guide](<./Video_Streaming_User_Guide.md> "Video Streaming User Guide").
