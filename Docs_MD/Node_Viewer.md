# Node Viewer

A [node](<./Node.md> "Node") can show either its icon or its viewer. Click the top-left node flag (the [Viewer Flag](<./Viewer_Flag.md> "Viewer Flag")) to toggle the viewer or icon. 

The viewer is only interactive when the node is in [Viewer Active](<./Viewer_Active.md> "Viewer Active") mode. 

left: node with 3-letter icon. middle: node with viewer (top-left flag of node). right: node with viewer active (bottom right flag). 

To get a floating resizable node viewer, right-click on the node and select View... 

Node viewers can be on by default when you create new nodes. To do this, turn on the **Viewer On by Default** option via the checkbox in Edit > [Preferences](<./Dialogs-Preferences_Dialog.md> "Dialogs:Preferences Dialog") > Network. 

Each operator family has its specific style of [Viewer](<./Viewer.md> "Viewer"). 

**Note** : Moving content inside a node viewer does not affect its transforms or parameters. **Exception** : [Camera COMP](<./Camera_COMP.md> "Camera COMP")
