# TUIO

The TUIO protocol provides a general and versatile communication interface between controller interfaces and applications. The [TUIO In DAT](<./TUIO_In_DAT.md> "TUIO In DAT") can receive TUIO messages from other applications on the network and parse them into columns. The protocol is encoded using the [OSC](<./OSC.md> "OSC") format which also allows TouchDesigner to get raw message data from the specification using the [OSC In CHOP](<./OSC_In_CHOP.md> "OSC In CHOP") or [OSC In DAT](<./OSC_In_DAT.md> "OSC In DAT"). 

Details about the TUIO specification can be found at [www.tuio.org](<http://www.tuio.org>)

## Examples

The [TUIO specification](<http://www.tuio.org/?specification>) includes profiles for 2D, 3D, and custom interactive surfaces. 

For example applications, see [Nuigroup](<http://nuigroup.com/log>)'s [Community Core Vision (CCV)](<http://nuicode.com/projects/tbeta>) vision-based tracking software and a list of other [TUIO software implementations here](<http://www.tuio.org/?software>).
