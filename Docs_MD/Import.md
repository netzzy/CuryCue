# Import

TouchDesigner can import a variety of media and data file types. See [File Types](<./File_Types.md> "File Types") for the list of supported file types and what [Operators](<./Operator.md> "Operator") they are converted to. 

You can import files by drag and dropping them to a TouchDesigner network. 

On the menu bar, import via File -> Import File... using the [Import File Dialog](<./Import_File_Dialog.md> "Import File Dialog")

Geometry and scene files in [FBX](<./FBX.md> "FBX") .fbx format are also imported through File -> Import File..., which will convert the .fbx files into the appropriate [Object](<./Object.md> "Object") Components and [SOP](<./SOP.md> "SOP") Operators. You can choose to import the camera, light and other data from the original file.
