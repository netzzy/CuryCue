# Keyframe

In the [Animation component](<./Animation_COMP.md> "Animation COMP") each keyframe specifies a channel's value at a specific time (or frame). A keyframe holds a value, incoming/outgoing slopes and accelerations, and an interpolation type. 

A channel's keyframes are used to interpolate and determine the values of all the samples of each channel that is output from the [Animation COMP](<./Animation_COMP.md> "Animation COMP"). 

See also [Animation Editor](<./Animation_Editor.md> "Animation Editor") and [Animation COMP](<./Animation_COMP.md> "Animation COMP").
