# Script

There are two scripting languages in TouchDesigner: [Python](<./Python.md> "Python") and the original [Tscript](<./Tscript.md> "Tscript"). Scripts and single-line commands can also be run in the [Textport](<./Textport.md> "Textport").   
  
Scripts are normally stored in Text or Table DATs. Python commands and Tscript commands can also be run directly via the [Textport](<./Textport.md> "Textport").
