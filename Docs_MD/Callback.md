# Callback

Some operators have a DAT [docked](<./Docking.md> "Docking") to them that contain some python functions. These functions, called "callbacks", get called when something in the operator changes. 

Examples are the Execute [DATs](<./DAT.md> "DAT"), [Timer CHOP](<./Timer_CHOP.md> "Timer CHOP"), [Replicator COMP](<./Replicator_COMP.md> "Replicator COMP")
