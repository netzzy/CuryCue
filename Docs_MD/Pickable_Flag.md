# Pickable Flag

The Pickable Flag is used for geometry selection in [3D Geometry Viewers](<./Geometry_Viewer.md> "Geometry Viewer") and [SOP Editors](<./SOP_Editor.md> "SOP Editor"). Geometry can only be selected when the Pickable Flag is on. The Pickable Flag is orange and only found on [Object COMPs](<./Object.md> "Object"). 

See also [Flag](<./Flag.md> "Flag").
