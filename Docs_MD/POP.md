# POP

##   
  
Summary

POPs (**Point Operators**) is a new [Operator Family](<./Operator_Family.md> "Operator Family") of TouchDesigner that runs on the GPU accelerated graphics card or chips, and creates/modifies 3D data which is rendered by the Render TOP or passed to devices like DMX lighting, LED arrays, lasers or other external systems. 

**The primary introductory documentation** for POPs and the POPs Example files are here: [Learning About POPs](<https://www.notion.so/touchdesigner/Learning-About-POPs-3f7645a368a043f99cd143e2382b8ab0?source=copy_link>)

Run Help -> OP Snippets in TouchDesigner to see examples per-POP type. 

See also [Category:POPs](<./Category-POPs.md> "Category:POPs") for a full list of articles related to POPs.
