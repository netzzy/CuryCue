# DMX

DMX and its derivatives are supported in the [DMX Out POP](<./DMX_Out_POP.md> "DMX Out POP"), [DMX Fixture POP](<./DMX_Fixture_POP.md> "DMX Fixture POP"), [DMX Out CHOP](<./DMX_Out_CHOP.md> "DMX Out CHOP") and [DMX In CHOP](<./DMX_In_CHOP.md> "DMX In CHOP").   
  
[sACN](<./SACN.md> "SACN") (Streaming Architecture for Control Networks) is a DMX protocol over IP networks using UDP. 

[Art-Net](<./Art-Net.md> "Art-Net") is an implementation of DMX512-A protocol over UDP. 

FTDI (serial DMX) is also supported. FTDI Driver version 2.12.16 

[KiNET](<https://www.colorkinetics.com/global/learn/optics-matter>) is the network protocol used by Philips Color Kinetics fixtures to receive DMX. It is very similar to ArtNET and sACN.
